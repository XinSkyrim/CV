## Prompt 1: Dataset Processing Pipeline

**Prompt:**
"Generate a Python pipeline to match blurred and sharp images by filename and prepare a dataset for object detection training."

**AI Output Summary:**
- Suggested using filename matching between blur and sharp directories
- Proposed using pandas DataFrame to store image pairs
- Provided code for building matched dataset structure

**Modifications Made:**
- Adapted code to use COCO dataset format
- Added filtering step to ensure images exist in COCO annotations JSON
- Integrated blur-level estimation for stratified splitting

## Prompt 2: COCO Label Extraction

**Prompt:**
"How to extract labels from COCO JSON using image filename instead of image ID?"

**AI Output Summary:**
- Suggested mapping file_name → image_id → annotations
- Provided dictionary-based lookup implementation

**Modifications Made:**
- Optimised lookup using hash maps
- Added bounding box conversion from COCO to YOLO format
- Ignored 'iscrowd' annotations for cleaner training
